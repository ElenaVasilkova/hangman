# Игра "Висилица"

"Висилица" - консольная игра. Игрок должен угадать загаданное слово, 
предлагая буквы или слово целиком и допустить не более 6 ошибок.

## Особенности реализации
- Угадывание слов русского языка
- Визуальное отображение виселицы
- Отслеживание использованных букв и слов
- Возможность повторной игры
- Автоматический резервный список слов
- Полная валидация ввода


## Установка и запуск

### Требования
Python 3.6 или выше

## Правила игры

- Компьютер загадывает случайное слово
- Игрок пытается угадать слово, вводя буквы или слова целиком
- При неверной букве/слове теряется попытка и рисуется виселица
- Игра продолжается до угадывания слова или исчерпания 6 попыток

## Формат файла words.txt

- Одно слово на строку  
- Только русские буквы  
- В формате UTF-8

Пример:

```text
яблоко
компьютер
программирование
телефон
```

## Функциональность
### Основные функции

`get_word()` - получение случайного слова  
`validate_letter()` - проверка введенной буквы  
`validate_word()` - проверка введенного слова  
`display_word_progress()` - отображение прогресса угадывания  
`draw_gallows()` - отрисовка виселицы  
`game()` - основной игровой цикл  

### Валидация ввода

- Проверка на русский алфавит
- Проверка повторного использования букв/слов
- Валидация ответов "да/нет"
- Обработка пустого ввода

### Пример игровой сессии
```
Давай поиграем в "Виселицу"? (да / нет)
да

У тебя 6 попыток.

*****
Введи одну букву или слово целиком.
а
Буква А есть в этом слове!
```

## Отладка

Если игра не находит файл words.txt, она автоматически использует встроенный список слов.
