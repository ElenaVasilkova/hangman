# Игра "Виселица"

"Виселица" - консольная игра. Игрок должен угадать загаданное слово, 
предлагая буквы или слово целиком и допустить не более 6 ошибок.

## Особенности реализации
- Угадывание слов русского языка
- Визуальное отображение виселицы
- Отслеживание использованных букв и слов
- Возможность повторной игры
- Автоматический резервный список слов
- Полная валидация ввода
- Модульная архитектура


## Установка и запуск

### Требования
Python 3.6 или выше

### Клонирование удаленного репозитория

```bash
git clone git@github.com:ElenaVasilkova/hangman.git
cd hangman/
```
### Создание виртуального окружения и установка зависимостей
```bash
python3 -m venv venv
source venv/bin/activate
python3 -m pip install --upgrade pip
pip install -r requirements.txt
```

### Запуск игры
```bash
python hangman.py
```

## Правила игры

- Компьютер загадывает случайное слово
- Игрок пытается угадать слово, вводя буквы или слова целиком
- При неверной букве/слове теряется попытка и рисуется виселица
- Игра продолжается до угадывания слова или исчерпания 6 попыток

## Формат файла words.txt

- Одно слово на строку  
- Только русские буквы  
- В формате UTF-8

Пример:

```text
яблоко
компьютер
программирование
телефон
```

## Архитектура проекта

### Модули и их ответственность

**main.py** - запуск приложения
**game.py** - основной игровой цикл, логика игры
**validators.py** - валидация ввода (буквы, слова, ответы)
**display.py** - отрисовка виселицы
**constants.py** - хранение всех текстов и настроек

### Основные функции

`get_word()` - получение случайного слова (words.py)    
`validate_letter()` - проверка введенной буквы (validators.py)  
`validate_word()` - проверка введенного слова (validators.py)  
`display_word_progress()` - отображение прогресса угадывания (game.py)  
`draw_gallows()` - отрисовка виселицы (display.py)  
`run_game()` - основной игровой цикл (game.py)  

### Валидация ввода

- Проверка на русский алфавит
- Проверка повторного использования букв/слов
- Валидация ответов "да/нет"
- Обработка пустого ввода

### Пример игровой сессии
```
Давай поиграем в "Виселицу"? (да / нет)
да

У тебя 6 попыток.

*****
Введи одну букву или слово целиком.
а
Буква А есть в этом слове!
```

## Отладка

Если игра не находит файл words.txt, она автоматически использует встроенный список слов.
